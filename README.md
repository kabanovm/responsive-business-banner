## Responsive Business Banner

A small React + TypeScript project implementing a responsive banner
with reusable UI components and accessibility best practices

https://responsive-business-banner.vercel.app/


### Tech

- React 19
- TypeScript
- Vite
- Vitest
- CSS Modules + BEM
- ESLint + Prettier
- clsx
- SVGR

### Setup & Run

```bash
pnpm install
pnpm run dev
```

### Testing
```bash
pnpm run test      
pnpm run test:ui # vitest UI in browser
```
